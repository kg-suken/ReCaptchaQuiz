# ReCaptcha風のクイズメイカー

ReCaptcha風のクイズを作成し、プレイできるサイトが作成できます。  
PHP環境をセットアップしてphpファイルをすべてアップロードしてください。  
クイズを作成できない場合はディレクトリに書き込み権限があるかを確認してください  
/questions/Debian　はサンプルです
